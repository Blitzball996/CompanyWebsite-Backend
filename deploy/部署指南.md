# 部署到香港服务器（保姆级 · 大白话）

> 目标：让 https://blitzball.lol 在公网能打开，自带 HTTPS🔒。
> 全部装在你那一台香港服务器上，一个域名搞定。你只需要复制粘贴命令。

---

## 准备：先做两件事（在你电脑上点点鼠标）

### 1) 把域名指向服务器
登录你买 `blitzball.lol` 的网站 → 找到「域名解析 / DNS」→ 添加两条记录（`服务器IP` 换成你腾讯云那台机器的公网 IP）：

| 主机记录 | 类型 | 记录值 |
|---------|------|--------|
| `@`     | A    | 服务器IP |
| `www`   | A    | 服务器IP |

### 2) 打开服务器防火墙的 80 和 443 端口
腾讯云控制台 → 轻量应用服务器 → 你的实例 → 「防火墙」→ 添加规则，放通 **80（HTTP）** 和 **443（HTTPS）**。

---

## 部署：在服务器上敲命令

### 3) 连上服务器
腾讯云控制台 → 轻量应用服务器 → 点「登录」→ 浏览器里会弹出一个黑色终端窗口（你已经是 root，不用输密码）。

### 4) 下载代码
> 两个仓库目前是 **私有(Private)**。最省事的做法是把它俩在 GitHub 上改成 **Public**（你的网站本来就是给全世界看的，后端也没存任何密码——密码是部署时现场随机生成的）。改成 Public 后，下面命令直接能用。
> 如果想保持私有，clone 时会让你输 GitHub 用户名 + Token（不是登录密码，是 GitHub → Settings → Developer settings → Personal access tokens 生成的、勾选 repo 权限的 token）。

复制粘贴（一次一段，回车）：

```bash
apt-get update -y && apt-get install -y git
cd /root
git clone https://github.com/Blitzball996/CompanyWebsite-Backend.git
git clone https://github.com/Blitzball996/Blitzball-Web.git CompanyWebsite
```

> ⚠️ 注意：前端那条命令结尾的 `CompanyWebsite` 不能改，部署脚本就找这个名字的文件夹。

### 5) 一键部署
```bash
cd /root/CompanyWebsite-Backend
bash deploy/deploy.sh blitzball.lol
```

这个脚本会自动：装 Docker → 生成随机强密码 → 编译启动后端+数据库+Caddy → 自动申请 HTTPS 证书。

**屏幕上会打印一个面板密码，截图/记下来！** 长这样：
```
DASHBOARD LOGIN:
  URL : https://blitzball.lol/dashboard
  user: admin
  pass: 一串随机字符   ← 记住这个
```

### 6) 等 30 秒，打开网站
手机或电脑浏览器输入 **https://blitzball.lol** —— 网站就上线了，地址栏带小锁🔒。

## 改了代码想更新上线

在你电脑上 `git push` 后，到服务器终端：
```bash
cd /root/CompanyWebsite-Backend && git pull && cd /root/CompanyWebsite && git pull
cd /root/CompanyWebsite-Backend && docker compose -f docker-compose.prod.yml up -d --build
```

## 出问题怎么查

```bash
cd /root/CompanyWebsite-Backend
docker compose -f docker-compose.prod.yml ps              # 看3个容器在不在
docker compose -f docker-compose.prod.yml logs -f caddy   # 看HTTPS证书日志
docker compose -f docker-compose.prod.yml logs -f server  # 看后端日志
```

**网站打不开/没有小锁** 最常见两个原因：
1. 域名解析还没生效（等几分钟）或 IP 填错；
2. 防火墙 80/443 没放通（回去看准备第2步）。

## 安全说明（已经帮你做好的）
- ✅ HTTPS 自动开启、自动续期（Caddy）
- ✅ 数据库和后端**不暴露公网**，只有 Caddy 对外（80/443）
- ✅ 面板密码是**随机生成**的强密码（不是默认的 change-me-please）
- ✅ 访客 ID 加盐哈希；`STORE_IP=true` 才存明文IP（隐私政策记得告知访客）
